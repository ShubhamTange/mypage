Project Details

E-Commerce Website Testing

Description: Developed a comprehensive automation testing suite using Java, Selenium WebDriver, TestNG, Jenkins, and Extent Reports for an e-commerce website. This project includes test cases covering critical functionalities such as product search, add to cart, and checkout processes. Jenkins integration ensures continuous testing and reporting, while Extent Reports provide detailed test execution summaries and metrics.

Details
  • Start Date: March, 2024
  • End Date: April, 2024
  • Technologies Used: Java, Selenium, TestNG, Git, Jenkins
  • Contributors: Shubham Tange
Objective
  • Develop a robust automation testing framework using Java, Selenium WebDriver, TestNG, Jenkins, and Extent Reports.
  • Automate test cases for critical functionalities of an e-commerce website: product search, add to cart, and checkout processes.
  • Ensure efficient and reliable testing to enhance software quality.
  • Accelerate release cycles through automated testing and continuous integration with Jenkins.
  • Provide detailed test execution summaries and metrics using Extent Reports.
